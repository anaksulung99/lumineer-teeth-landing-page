# Project Blueprint

## Overview

This project is a Next.js landing page for Lumineer Teeth. It is designed to be a responsive, single-page application that is easy to update and maintain.

## Design and Features

### Style

*   **Colors:** The color scheme is based on a cyan, white, and slate color palette.
*   **Typography:** The primary font is Inter.
*   **Icons:** The application uses icons from the `lucide-react` library.
*   **Layout:** The layout is a single-column layout that stacks on mobile devices.

### Features

*   **Hero Section:** A hero section with a headline, description, and call-to-action button.
*   **Story Section:** A section that tells the story of the product.
*   **Benefits Section:** A section that highlights the benefits of the product.
*   **Pricing Section:** A section that displays the pricing of the product.
*   **Testimonials Section:** A section that displays testimonials from customers.
*   **FAQ Section:** A section that answers frequently asked questions.
*   **Sticky CTA:** A sticky call-to-action button that is always visible on the screen.
